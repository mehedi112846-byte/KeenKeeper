# 👥 KeenKeeper — Keep Your Friendships Alive

**KeenKeeper** is a sophisticated friendship management dashboard designed to help users maintain meaningful connections in a fast-paced world. By tracking interactions, setting contact goals, and visualizing social habits, KeenKeeper ensures no friend is left behind.

---

---

## 🚀 Key Features

* **Smart Friend Tracking:** A comprehensive dashboard showcasing friend statuses—`overdue`, `almost due`, or `on-track`—with visual cues to prioritize who needs a call.
* **Interactive Timeline:** Automatically logs every call, text, or video chat. Users can filter their interaction history to see how they've been communicating over time.
* **Friendship Analytics:** Integrated data visualization using **Recharts** that displays a breakdown of interaction types (Call vs. Text vs. Video) through an intuitive Pie Chart.
* **Responsive Experience:** Fully optimized for mobile, tablet, and desktop screens with a modern, Figma-accurate UI.
* **Quick Check-ins:** One-click actions to log interactions with instant toast notifications and real-time timeline updates.

---

## 🛠️ Technologies Used

- **Frontend:** React.js
- **Styling:** Tailwind CSS & Lucide React (for Icons)
- **Routing:** React Router DOM 
- **Data Visualization:** Recharts
- **Notifications:** Toastify
- **Deployment:** Vercel / Netlify

## 📋 Project Requirements Met

### 1. Dynamic Friend Management
- Fetches realistic friend data from a structured JSON source.
- Status-based styling for friend cards (Overdue, Almost Due, On-track).
- Detailed profile pages with bio, contact goals, and relationship stats.

### 2. Activity Logging (Timeline)
- Real-time logging of interactions.
- **Filter Support:** Users can filter the timeline by interaction category (Call, Text, Video).

### 3. Analytics Dashboard
- High-level summary cards on the homepage.
- Pie chart representation of friendship statistics on the Stats page.

### 4. Robust UX/UI
- Custom 404 error page for invalid routes.
- Smooth loading animations during data fetching.
- Fully responsive grid layouts (4-column layout for large screens).
